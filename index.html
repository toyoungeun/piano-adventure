🎹 피아노 연결 방법 (야마하 CLP-835)

방법 1: USB 케이블 (가장 확실해요)

피아노 아래쪽의 [USB TO HOST] 단자와 컴퓨터를 USB 케이블(B타입)로 연결한 뒤, 이 페이지를 Chrome 브라우저로 열고 위의 "피아노 연결하기" 버튼을 누르세요. 브라우저가 MIDI 사용 권한을 물어보면 허용을 눌러주세요.

방법 2: 블루투스 (Mac)

Mac에서 오디오 MIDI 설정 앱 → 윈도우 메뉴 → 블루투스 구성 보기를 열고 CLP-835를 연결한 다음, "피아노 연결하기" 버튼을 누르세요.

윈도우에서 피아노가 인식되지 않을 때

보통은 USB만 꽂으면 바로 인식되지만, 안 될 경우 야마하 공식 드라이버를 설치하세요: Yamaha USB-MIDI Driver 다운로드 (Windows 10/11)

바탕화면에 게임 아이콘 만들기

Chrome으로 이 게임을 연 뒤, 주소창 오른쪽 설치 아이콘(⊕ 모양) 또는 메뉴(⋮) → 전송, 저장, 공유 → 페이지를 앱으로 설치를 누르면 바탕화면에 🎹 아이콘이 생기고 앱처럼 실행할 수 있어요.

연결이 안 되어도 괜찮아요

화면 건반을 마우스나 터치로 눌러서 모든 게임을 할 수 있어요.
